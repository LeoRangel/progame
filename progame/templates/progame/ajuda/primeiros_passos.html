Introdução #

Bem vindo a central de ajuda do ProGame! Aqui você poderá encontrar todas as informações a respeito de como usar a plataforma.

Use o menu de opções lateral para navegar entre as seções da página e se quiser voltar para a página inicial da central de ajuda, clique em "Central de ajuda" no cabeçalho da página.

Para começar a usar a plataforma você precisa, primeiramente, criar uma conta como professor ou uma conta como aluno. Siga para a seção de Cadastro para aprender como registrar-se no sistema e se já fez isso, você pode fazer Login.

Criar uma conta #

Para criar uma conta no ProGame é simples:


1 - Vá para a página de cadastro clicando no botão Registrar-se na barra de navegação localizada no topo da página.

2 - Escolha uma das formas de cadastro: Registrar-se com a conta do google ou Preencher o formulário.

3 - Se você escolheu Registrar-se com a conta do google, tudo o que precisa fazer é clicar no botão com esse mesmo nome e escolher a conta que será utilizada. Se você prefere a segunda opção, preencha o formulário com um endereço de email e uma senha válidos (obs: no 3º campo do formulário você precisa repetir a senha como procedimento de segurança) e clique no botão Continuar.

O próximo passo de criar uma conta é escolher o seu tipo de usuário.

Escolher tipo de usuário #

Após fazer a primeira parte do cadastro, você precisa agora escolher o tipo de usuário que você será no sistema, para finalizar o procedimento e ser redirecionado para a página de inicial do usuário. Para isso, realize os seguintes passos:


1 - Leia a descrição e escolha uma das opções de usuário: Aluno ou Professor.

2 - Clique na opção escolhida.

3 - Clique no botão Continuar.


Feito isso, você será redirecionado para a página inicial do usuário Minhas turmas.

Entrar na minha conta #

Para fazer login no sistema, você precisa ter criado uma conta anteriormente. Se ainda não fez esse procedimento, volte para a seção cadastro e veja o passo a passo de como você pode fazer isso. Caso já tenha uma conta no ProGame, você pode fazer login da seguinte maneira:


1 - Clique no botão Entrar localizado na barra de navegação no topo da página para ser redirecionado para a página de login.

2 - Preencha o formulário de login com email e senha da sua conta.

3 - Clique no botão Entrar.


Feito isso, você será redirecionado para a página inicial do usuário Minhas turmas ou para a página de Escolher tipo de usuário no caso de o cadastro da conta ainda não está finalizado.

Ver perfil #

Para ver o seu perfil de usuário é simples.


1 - Na barra de navegação localizada no topo da página, clique na opção que mostra sua foto e nome de usuário.

2 - Localize a opção Meu perfil e clique para que seja aberta a tela de perfil.

Editar dados pessoais #

Para editar os seus dados de usuário, realize os seguintes procedimentos:


1 - Na barra de navegação localizada no topo da página, clique na opção que mostra sua foto e nome de usuário.

2 - Localize a opção Editar perfil e clique para que seja aberta a tela de edição de perfil.

3 - Realize as edições necessárias.

4 - Clique no botão Salvar.
